    • Outline VPN 搭建

    Outline 是 Jigsaw 创立的开源项目，它由 Shadowsocks 提供支持，实际上它的目标用户是新闻机构，为了保护机构内部通讯隐私。

    它提供了几种简易的安装方式，支持 DigitalOcean、GCP、AWS 或其他提供商。

    当然首先你需要一台服务器，本文使用的环境为 Ubuntu 18.04。

    你可以在执行完脚本之后的错误日志中找到 Outline 使用的端口号，同时在 Outline Manager 设置好的服务器中，在服务器信息一节可以找到用于新访问密钥的接口，仅需要开放这几个端口即可。
